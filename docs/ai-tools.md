# Use with AI tools

In order to use AI tools, e.g., agents most effectively with your Calkit
project, you should first ensure there is an `AGENTS.md` file at the
root.
This should explain the project's goals and workflow.
For example:

```md
# Agent instructions

In this project we are trying to answer the research questions in `calkit.yaml`.

In general, we're running simulations with Julia,
visualizing data in notebooks, and writing our paper with LaTeX.

It is important that all derived artifacts are created as part of the
Calkit pipeline so their provenance is fully defined.
```
